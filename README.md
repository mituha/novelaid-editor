# novelaid-editor

WEB小説執筆支援のための拡張可能なデスクトップエディタです。Electron, React, TypeScript で構築されています。
強力なAI支援と、Gitによる履歴管理を統合し、快適な執筆環境を提供します。

## はじめに

このプロジェクトは、作者が自身のWEB小説執筆のために開発しています。
また、バイブコーディング（生成AIによる対話型プログラミング）で開発を進めるための実験的なプロジェクトでもあります。
そのため、As is で公開されています。
本READMEおよび `docs/` 下のドキュメント類もバイブコーディングによって生成されているため、**未実装の内容や、実際の実装とは異なる記述が含まれる可能性があります**。
ファイル表示、エディター、ローカルAIアシスタントの雛形ぐらいにはなるかもしれません。

---

## 📖 執筆者の方へ

novelaid は、物語を紡ぐことに集中するための機能を備えています。

- **書庫（プロジェクト）管理**: フォルダを一つの「書庫」として管理。ライブラリ画面から最近開いた作品にすぐ戻れます。
- **安心の履歴管理 (Git)**: 執筆した内容の変更履歴を自動的・明示的に保存。過去のバージョンを確認したり、特定の時点に戻したりすることが可能です。
- **AIアシスタント**: 執筆の相談、プロットの練り直し、校正などを、エディタ内のサイドパネルからAI（Gemini, OpenAI, LM Studio）に依頼できます。
- **執筆に特化したUI**: 画面分割、文字数リアルタイムカウント、ダークモードなど、長時間の執筆を支えるシンプルなインターフェース。

## 💻 開発者の方へ

拡張性とモダンな技術スタックを重視して設計されています。

- **技術スタック**: Electron / React 19 / TypeScript / Monaco Editor
- **プラグイン機構**: 設定画面へのタブ追加など、機能をモジュール化して拡張できる仕組みを備えています。
- **AIプロバイダー抽象化**: 新しいAIモデルやAPIプロバイダーを容易に追加できるインターフェース設計。

## 🛠 開発環境のセットアップ (Windows / PowerShell)

開発には Windows 環境と PowerShell (`pwsh`) を推奨しています。

### 1. 前提条件の確認
- **Node.js**: v18以上が必要です。
- **npm**: Node.jsに付属しているパッケージマネージャ。

### 2. インストール
リポジトリをクローンし、必要なパッケージをインストールします。

```pwsh
# リポジトリの取得
git clone https://github.com/mituha/novelaid-editor.git

# フォルダへ移動
cd novelaid-editor

# 依存パッケージのインストール
npm install
```

### 3. 開発モードでの起動
プログラムを起動して、変更をリアルタイムに確認しながら開発できます。

```pwsh
# 開発サーバーとアプリの起動
npm start
```

### 4. パッケージング
配布用の実行ファイルを作成します。

```pwsh
# アプリのパッケージ化
npm run package
```
*※Windows環境ではシンボリックリンク作成の権限不足でエラーになる場合があるため、その際は「管理者として実行」したターミナルで実行してください。*
