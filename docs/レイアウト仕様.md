# レイアウト仕様

画面、レイアウト構成に関する仕様、および、メモ。

## メインウィンドウ

固定配置の以下の領域に分かれます。

* 上部：メインメニュー
* 中央
    + 左側：左ペイン
    + 中央：ドキュメントビューエリア
    + 右側：右ペイン
* 下部：ステータスバー

これらの領域は表示有無の切り替えはありますが、配置は固定とします。

### メインメニュー

### 左ペイン

作業の起点となるファイルの一覧を表示する領域
ここで選択したドキュメントがドキュメントエリアに表示されます。

* ファイルエクスプローラー
* 登場人物一覧等
* Git変更ステータス

### 右ペイン

ドキュメントビューリアで編集している内容に対する処理や情報表示を行う領域

* AIチャット
* AI校閲

### ドキュメントエリア

編集するドキュメントを表示する領域
左右分割表示が可能。
