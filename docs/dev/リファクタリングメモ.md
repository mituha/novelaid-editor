# リファクタリングメモ

バイブコーディングで書かせているため、あちこち問題がある場合の修正用のメモ。




## チャット表示の共通化

現在、AIチャットのメッセージ表示部分として、

* ChView
* AIChatPanel
* AIProofreaderPanel

の三箇所があります。
このメッセージ表示部分をChViewの仕様に統一し、共通化したコンポーネントに変更します。
また、内部的にはDocumentContextを介してファイル(ドキュメント)を参照する形とします。
この際、AIChatPanelとAIProofreaderPanelでは、.novelaid以下に一時ファイルを作成して対応するものとします。
これにより、表示内容の保持も行えるようになります。
