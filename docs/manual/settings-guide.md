# 設定ガイド

自分の好みに合わせてエディターをカスタマイズします。

## エディター設定

- **フォントサイズ**: 執筆しやすい文字の大きさに調整できます。
- **ワードラップ**: 画面端で自動的に折り返すかどうかを設定します。
- **デフォルト拡張子**: 新しいファイルを作成した際の標準的な拡張子（.md / .txt）を選択します。

## 校正（キャリブレーション）設定

textlintを利用した文法チェックの設定を切り替えられます。
- ら抜き言葉のチェック
- 二重助詞のチェック
- 全角・半角スペースのルール

## テーマ設定

ダークモード・ライトモードの切り替えが可能です。
