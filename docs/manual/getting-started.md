# クイックスタート

Novelaid Editorを使って執筆を開始するまでの手順を説明します。

## 導入方法（ダウンロード）

最新のアプリは以下のGitHubリリースページからダウンロードできます。

**[最新リリースのダウンロード](https://github.com/mituha/novelaid-editor/releases/latest)**

### インストール手順（Windows）

1. 上記リンク先の **Assets** セクションから、`.exe` ファイル（例: `novelaid-editor-setup-x.x.x.exe`）をダウンロードします。
2. ダウンロードしたファイルを実行します。
3. **インストール時の警告について**:
   本アプリは署名のないインストーラーを使用しているため、Windows SmartScreenにより「WindowsによってPCが保護されました」という警告が表示されることがあります。その場合は、**「詳細情報」**をクリックし、表示された**「実行」**ボタンを押すことでインストールを継続できます。

## プロジェクトの開始

アプリを起動すると「ライブラリ」画面が表示されます。

### 既存のフォルダを開く
すでに執筆中のフォルダがある場合は「既存のフォルダを開く」から選択します。

### 新しいプロジェクトを作成
新しく執筆を始める場合は「新しいプロジェクトを作成」を選択し、保存先のフォルダを指定します。
- プロジェクト名を入力すると、その名前のフォルダが作成されます。
- Gitリポジトリとして初期化され、履歴管理が可能な状態で開始されます。

## メイン画面の構成

執筆画面は大きく4つのパーツで構成されています。

1. **左サイドバー**: ファイル操作やGit操作の切り替え。
2. **中央エディター**: メインの執筆エリア。
3. **右パネル**: AIチャットや校閲機能。
4. **ステータスバー**: 文字数確認や設定へのアクセス。
