# Gitによる履歴管理

Novelaid Editorは、執筆の過程をGitによって詳細に記録します。

## コミット（履歴の保存）

左サイドバーのGitアイコン（枝分かれした線）をクリックすると、現在の変更内容が表示されます。

1. **変更の確認**: 前回の保存時からの差異が色分け（赤が削除、緑が追加）で見ることができます。
2. **コミットメッセージの入力**: 「第1章完成」などの変更内容を入力します。
3. **コミットボタンのクリック**: これで現在の状態が履歴に固められます。

## コミットグラフ

過去の執筆履歴をグラフ形式で視覚的に確認できます。過去の特定の時点に遡ったり、執筆のペースを振り返るのに役立ちます。
