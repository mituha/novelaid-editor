# AIアシスタントの活用

AIを活用して、アイデア出しや原稿のブラッシュアップが可能です。

## AIチャット

右パネルのコメントアイコンから、AIと対話できます。
- 「ここから先の展開を一緒に考えて」
### APIキーの設定

Geminiを使用する場合、設定画面からAPIキーを入力するか、システム環境変数 `GOOGLE_API_KEY` を設定することで利用を開始できます。

## 登場人物（キャラクター）との対話

設定したキャラクターとしてAIに振る舞わせることができます。
- 「このシーンのあなたの行動、口調に違和感がないか見て」
- 「あなたの生い立ちについてもっと詳しく教えて」
といった相談が可能です。
機能の詳細は[AIチャット詳細ガイド](ai-chat.md)を参照してください。

## AI校閲

星マークのアイコンから、選択中の範囲の文章を校正できます。
- **校正の依頼**: 誤字脱字の修正や、表現のブラッシュアップを依頼します。
- **思考の表示**: AIがどのように考え、なぜその修正を提案したのかの過程を見ることができます。

## AIの設定
AI機能を利用するには、設定画面からAPIキーなどの設定が必要です。
- **Gemini**: Google提供の強力なAI。設定画面での入力のほか、環境変数 `GOOGLE_API_KEY` からも取得可能です。
- **OpenAI**: ChatGPT等で知られるAI。
- **LM Studio**: ローカル環境でAIを動かしたい場合に利用します。
