# 小説執筆エディター マニュアル

## 概要
このエディターは、直感的なインターフェースに執筆管理・Git履歴・AI支援を統合した、小説執筆のための専用ツールです。

## 基本的な操作

### 1. ライブラリ画面 (書庫の選択)
アプリ起動時は「ライブラリ」画面が表示されます。
- **既存のフォルダを開く**: 小説のプロジェクトフォルダを選択して読み込みます。
- **新しい書庫を作成**: 新しいフォルダを選択して執筆を開始します。
- **最近使った書庫**: 一度開いたプロジェクトはリストに保存され、次回からワンクリックで開けます。

### 2. エディタ画面
エディタ画面では以下のエリアに分かれています。
- **サイドバー (左)**: ファイル一覧、Git変更履歴、ログなどを切り替えて表示します。
- **エディタエリア (中央)**: 執筆を行うメインエリア。タブ表示や画面分割に対応しています。
- **サイドパネル (右)**: AIチャットやAI校閲などの支援機能を表示します。
- **ステータスバー (下)**: 現在の文字数カウント、設定、ライブラリ画面への移動ボタンがあります。

### 3. 保存
編集内容は **Ctrl + S** で保存できます。

## 便利な機能

### 文字数カウント
ステータスバーに、現在開いているファイルの文字数がリアルタイムで表示されます。

### Gitによる履歴管理
左サイドバーのGitアイコンから、現在の変更点を確認したり、コミット（履歴の確定）を行ったりできます。

### AIアシスタント
右側のAIアイコンから、AIとのチャットや原稿の校正が可能です。
※利用には設定画面から API キーやプロバイダー（Gemini, OpenAI, LM Studio）の設定が必要です。

## ショートカットキー
| キー操作 | 機能 |
| --- | --- |
| `Ctrl + S` | 保存 |
| `Ctrl + \` | エディタ画面の分割・解除 |
| ステータスバーの Home アイコン | ライブラリ画面へ戻る |
