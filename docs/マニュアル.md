# 小説執筆エディター マニュアル

## 概要
このエディターは、VSCodeライクなインターフェースで小説を執筆するためのツールです。
Monaco Editorを採用しており、快適な執筆体験を提供します。

## 使い方

### 1. フォルダを開く
サイドバーの「フォルダを開く」ボタンをクリックし、執筆プロジェクトのフォルダを選択してください。
フォルダ内のファイル一覧が表示されます。

### 2. ファイルを開く・編集する
サイドバーのファイル一覧から、編集したいファイルをクリックします。
右側のエディターにファイルの内容が表示されます。

### 3. ファイルを保存する
編集内容は、ショートカットキー **Ctrl+S** (または Cmd+S) で保存できます。

## 機能
- **ファイルエクスプローラー**: フォルダ内のファイルを一覧表示します。ディレクトリは上位に表示されます。
- **ダークモード**: 目に優しいダークテーマを採用しています。
- **日本語フォント**: 執筆に適した日本語フォント設定が適用されています。

## ショートカットキー
| キー操作 | 機能 |
| --- | --- |
| `Ctrl + S` | ファイルの上書き保存 |

## トラブルシューティング
- **ファイルが表示されない**: フォルダの読み込み権限がない可能性があります。別のフォルダを試してください。
- **保存できない**: ファイルが読み取り専用になっていないか確認してください。
